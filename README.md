# Avatars

A small, curated collection of neutral headshot images intended for use as **placeholder avatars** in web and application interfaces.

These images are designed to represent users where a real profile photo is unavailable or unnecessary.

## Intended Uses

This package is suitable for:

- Default user avatars in dashboards or admin panels  
- Seed / demo data for development and staging environments  
- UI mockups, prototypes, and design systems  
- Screenshots, documentation, and marketing previews  
- Multi-tenant or SaaS apps that need visually distinct but neutral user profiles  

## What This Is *Not*

- This is **not** a stock photography service  
- These images are **not** intended to represent real people or identities  
- This package does **not** handle uploads, cropping, or avatar generation
